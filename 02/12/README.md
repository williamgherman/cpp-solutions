### Exercise 2.12

Which, if any, of the following names are invalid?

(a) `int double = 3.14;`  
(b) `int _;`  
(c) `int catch-22;`  
(d) `int 1_or_2 = 1;`  
(e) `double Double = 3.14`

### Solution

`double` and `catch-22` are invalid identifiers because of the reserved keyword
`double` and use of the hyphen.
