### Exercise 2.07

What values do these literals represent? What type does each have?

(a) `"Who goes with F\145rgus?\012"`  
(b) `3.14e1L`  
(c) `1024f`  
(d) `3.14L`

### Solution

(a) "Who goes with F&aelig;rgus?", followed by a Form Feed; string literal
(`char[]`).  
(b) 31.4; `long double`  
(c) 1024.0; `float`  
(d) 3.14, `long double`
