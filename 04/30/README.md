### Exercise 4.30

Using table 4.12 (p. 166), parenthesize the following expressions to match the
default evaluation:

(a) `sizeof x + y`  
(b) `sizeof p->mem[i]`  
(c) `sizeof a < b`  
(d) `sizeof f()`

### Solution

(a) `(sizeof(x) + y)`
(b) `(sizeof((p->mem)[i]))`  
(c) `((sizeof(a)) < b)`  
(d) `(sizeof((f())))`
